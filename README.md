# 📊 Modelling Experimental & Observational Data in Python

> **MSc Module Project | University of Essex | Machine Learning | COVID-19 Analysis | 100% Classification Accuracy**

A comprehensive machine learning study analysing the relationship between World Development Indicators and COVID-19 death rates, using clustering, binary classification, and multi-class classification algorithms.

---

## 🎯 Project Overview

This project explores how global development indicators (healthcare investment, GDP, population density, etc.) relate to COVID-19 mortality rates across countries. Using unsupervised and supervised ML techniques, the analysis uncovers patterns across continents and builds predictive models for high COVID-19 casualty risk.

---

## 🔬 Key Results

| Model | Task | Accuracy |
|-------|------|----------|
| Logistic Regression | Binary Classification (High vs Low COVID casualties) | **100%** |
| Linear Discriminant Analysis (LDA) | Multi-class Classification | **89.3%** |
| Quadratic Discriminant Analysis (QDA) | Multi-class Classification | Compared |
| K-Means Clustering | Continent-level grouping | 6 clusters |

---

## 🧠 Technical Approach

### 1. Unsupervised Learning — K-Means Clustering
- Clustered countries into 6 groups corresponding to continental regions
- Identified development and mortality patterns across global clusters

### 2. Binary Classification — Logistic Regression
- Predicted high vs low COVID-19 casualty risk per country
- Achieved 100% accuracy on test data

### 3. Multi-Class Classification
- Compared three algorithms: Logistic Regression, LDA, and QDA
- LDA achieved highest accuracy of 89.3%
